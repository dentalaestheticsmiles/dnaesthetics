## How to Use

1. Open `index.html` in a web browser
2. The website is ready to use - no build process required!

## Customization

### Update Contact Information

Edit the following in `index.html`:
- Phone number: Replace `+1234567890` with your actual phone number
- Email: `dentalaestheticsmiles@gmail.com` (already configured)
- Address: Update the address in the Contact section
- Social Media Links: Update Instagram, Facebook, and WhatsApp links

### Update Images

Replace the Unsplash image URLs with your own clinic images:
- Hero section background
- Service cards images
- Tour gallery images
- Blog post images

### Update Services

Modify the services list in the Services section to match your actual offerings.
